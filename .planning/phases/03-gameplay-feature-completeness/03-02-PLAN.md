**Ghost piece verification:**
1. A dimmed/faded version of the current piece should appear at the bottom of the board showing where it will land
2. Move the piece left/right -- the ghost should update in real-time
3. Rotate the piece -- the ghost should reflect the new rotation
4. The ghost should be visually distinct from the active piece (dimmer)

**Hold piece verification:**
5. Press 'c' -- the current piece should go to the Hold panel (right side), and a new piece spawns
6. Press 'c' again -- the held piece should swap with the current piece
7. Press 'c' a third time before the swapped piece locks -- it should be REJECTED (no swap happens)
8. Let the piece lock (drop it), then press 'c' -- hold should work again

**Next piece verification:**
9. A "Next" panel should show the upcoming piece
10. When the current piece locks, the next piece that spawns should match what was shown in the preview

**Statistics verification:**
11. A statistics panel should be visible showing piece counts and/or line clear counts
12. Lock a piece -- the pieces counter should increment
13. Clear a line -- the appropriate line type counter (Single/Double/Triple/Tetris) should increment

**Pause verification:**
14. Press 'p' -- the game should freeze (piece stops falling, "PAUSED" text appears on board)
15. Try moving/rotating -- nothing should happen (keys blocked during pause)
16. Press 'p' again -- game should resume, piece continues falling at normal speed (no sudden drop)
17. Press 'q' or ESC during pause -- game should exit

**Regression check:**
18. Movement (arrow keys), rotation (up/z), soft drop (down), hard drop (space) all work
19. Line clearing works and score updates correctly
20. Game over triggers when board fills up
