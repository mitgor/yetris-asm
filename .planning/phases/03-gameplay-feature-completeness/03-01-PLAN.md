1. `_compute_ghost_y()` -- Pure query function that returns the lowest valid Y position for the current piece without modifying any game state. Uses callee-saved registers x19-x22 for piece state. Loop: try y+1, call `_is_piece_valid`, stop when invalid, return last valid y in w0. See research pattern "Ghost Piece Y Computation" for exact implementation. CRITICAL: This function must NEVER write to _piece_y or any other global. It reads _piece_type, _piece_rotation, _piece_x, _piece_y into registers and operates only on registers.

2. `_hold_piece()` -- Implements the hold mechanic. Algorithm:
   a. Load _can_hold; if 0, return immediately (already held this turn)
   b. Set _can_hold = 0
   c. Load current _piece_type into a register
   d. Load _hold_piece_type
   e. If _hold_piece_type == 0xFF (empty): store current type into _hold_piece_type, call _spawn_piece (gets next from bag), return
   f. Else (hold has a piece): swap -- store current type into _hold_piece_type, set _piece_type = old hold type, set _piece_rotation = 0, load spawn position from _piece_spawn_x and _piece_spawn_y tables for the swapped-in type, call _check_game_over

   Uses callee-saved x19 (current type), x20 (held type). Stack frame: 32 bytes.

**1. _draw_ghost_piece()** -- New function. Draw the ghost piece using A_DIM attribute:
   - Call _compute_ghost_y() to get ghost_y in w0
   - Load piece state (type, rotation, piece_x) and ghost_y
   - Load color pair from _color_pairs[type]
   - Iterate 5x5 grid (same loop structure as _draw_piece)
   - For each non-empty cell: compute screen_y = ghost_y + r + 1, screen_x = (piece_x + c) * 2 + 1
   - Skip if screen_y outside [1, 20] or screen_x outside [1, 20]
   - Set attribute: COLOR_PAIR(n) | A_DIM where A_DIM = 0x100000. Compute as: lsl w1, wN, #8 then orr w1, w1, #0x100000

   - NOTE: A_DIM 0x100000 cannot be encoded as a bitmask immediate in ORR. Use: mov w9, #0x10 / lsl w9, w9, #16 / orr w1, w1, w9 (or movz w9, #0x10, lsl #16 / orr w1, w1, w9)
